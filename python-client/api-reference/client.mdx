## BaseLiteralClient

```python
class BaseLiteralClient()
```

<Warning>Use LiteralClient or AsyncLiteralClient in your code. Both classes inherit from BaseLiteralClient and have access to its methods.</Warning>

**Attributes**:

<ResponseField name="api" type="Union[LiteralAPI, AsyncLiteralAPI]">Instance of the Literal API.</ResponseField>
<ResponseField name="disabled" type="bool">Flag indicating if the client is disabled.</ResponseField>
<ResponseField name="event_processor" type="EventProcessor">Processor for handling events.</ResponseField>
  

**Arguments**:

<ResponseField name="batch_size" type="int, optional">Batch size for event processing. Defaults to 5.</ResponseField>
<ResponseField name="is_async" type="bool, optional">Flag indicating if the API should be asynchronous. Defaults to False.</ResponseField>
<ResponseField name="api_key" type="Optional[str], optional">API key for authentication. If not provided, it will be fetched from the environment variable 'LITERAL_API_KEY'.</ResponseField>
<ResponseField name="url" type="Optional[str], optional">URL of the Literal API. Defaults to 'https://cloud.getliteral.ai'.</ResponseField>
<ResponseField name="disabled" type="bool, optional">Flag to disable the client. Defaults to False.</ResponseField>
  

**Raises**:

<ResponseField name="Exception">If 'LITERAL_API_KEY' is not provided and not found in environment variables.</ResponseField>
